# VitalIQ

A Health & Fitness Tracking app built with Flutter and Firebase.

> 🚧 Work in progress

## Features
- Google / Apple / Email Authentication
- Dashboard & Home Screen
- Workout Tracking
- Diet & Calorie Tracking
- AI Coach (Chat)
- Sleep Tracking
- Push Notifications

## Tech Stack
- Flutter & Dart
- Firebase (Auth, Firestore, Crashlytics)
- Riverpod (state management)
- GoRouter (navigation)


> Add your own `google-services.json` to `android/app/` before running.
